Page Structure

  • All pages are made out of a tree like structure. These are boxes, inside of boxes, inside of boxes.
  • There are h1, div, em, img.

Visual Styling

  • You can modify text size, colors, and other css using Chrome's Developer Tools.

HTML-CSS-DOM

  • Document Object Model (DOM) is the three in the html structure.

Boxes Everywhere

  • All pages are boxes.

Summary

The three structure, boxes, and in-line boxes work together to build the HTML structure of a web page. The three structure is the HTML displayed in an organized manner for a developer code. The box structure is the visual representation of the three. The in-line and boxes are the rules of how objects (divs) behave within the structures. Beyond the structure CSS can uniformly style similar objects.

Lecture 3: Adding CSS Style to HTML Structure

Understanding CSS

Cascading means that each style rule is applied to the a specific value, and its descendants.

CSS is about avoiding styling repetition. Avoiding repetition is important because it will save you time, help you avoid errors, and create a consistent design.

Styling up

CSS is very powerful and has many options to style a page. It's best to reference selectors by looking at the CSS documentation.

CSS-Trics article Beginner Concepts: How CSS Selectors Work is a great resource to remind yourself of CSS selectors options.

Proper use of terms

  • background-color: back; - is a deceleration
  • background-color - is a property
  • black - is the value

You can add style to a html document by insert CSS within style tags on the head of a document.

The Box Revisited

Here is a explanation of the box model.

The border can be difficult to get with so many layers. Instead use box-sizing

The CSS style below makes the size of the element include both the border and margin.

*{
box-sizing: border-box;
}

Note: this may not work on older browsers. To ensure it works on all browsers do the following.

*{
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
-ms-box-sizing: border-box;
box-sizing: border-box;
}
